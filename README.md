# Vrinda Store Data Analysis Dashboard

## Project Overview
This project presents a sales data analysis dashboard for Vrinda Store using Microsoft Excel.
The dashboard is built with pivot tables, pivot charts, and slicers to analyze sales trends and customer behavior.

## Tools & Technologies
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Data Cleaning & Formatting
- Slicers and Filters

## Key Insights
- Women contribute approximately 64% of total sales
- Amazon is the highest-performing sales channel
- Maharashtra generates the highest revenue among states
- 92% of orders are successfully delivered
